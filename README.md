# InAmigos Foundation Awareness Webpage

## Overview

This project is a responsive NGO awareness webpage developed using HTML, CSS, and JavaScript to showcase the mission, projects, initiatives, and social impact of InAmigos Foundation.

The webpage was created as part of an internship assignment focused on spreading awareness about the organization's efforts in education, community welfare, environmental sustainability, women empowerment, animal welfare, and skill development.

---

## Features

* Responsive and user-friendly design
* Modern landing page with hero banner
* NGO introduction and mission section
* Showcase of flagship projects
* Social impact statistics
* Image gallery
* Contact information section
* Social media links
* Smooth navigation and interactive UI
* Mobile-friendly layout

---

## Projects Highlighted

### Project SEVA

Community welfare initiatives focused on helping underserved communities through social outreach and humanitarian support.

### Project BACHPANSHALA

Educational initiatives aimed at supporting and empowering underprivileged children through learning opportunities.

### Project UDAAN

Programs dedicated to women empowerment, awareness, confidence building, and skill enhancement.

### Project JEEV

Animal welfare initiatives promoting care, compassion, rescue, and protection.

### Project PRAKRITI

Environmental sustainability campaigns including plantation drives and awareness activities.

### Project VIKAS

Skill development and growth-oriented initiatives designed to enhance employability and career opportunities.
